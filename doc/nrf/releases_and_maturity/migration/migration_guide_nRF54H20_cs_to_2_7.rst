Migration notes for |NCS| v2.7.0 for nRF54H20 DK users
######################################################

This document describes the new features and changes implemented in |NCS| to take into consideration when migrating your application for the nRF54H20 DK to |NCS| v2.7.0.

To ensure the nRF54H20 DK runs its components correctly, verify its compatibility with the |NCS| v2.7.0 release by checking the version number on your DK's sticker.
The only versions of the nRF54H20 DK compatible with the |NCS| v2.7.0 are the following:

* Version PCA10175 v0.7.x (ES3)
* Version PCA10175 v0.8.0 (ES3.x)
* Version ES4
